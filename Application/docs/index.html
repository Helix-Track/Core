Quick Start Guide

Get Started in 5 Minutes

Follow these steps to set up and run HelixTrack Core locally.

Prerequisites

# Required
- Go 1.22+
- SQLite 3 or PostgreSQL 12+
- Git

Installation

# Clone repository
git clone https://github.com/Helix-Track/Core.git
cd Core/Application

# Install dependencies
go mod download

# Run tests
./scripts/verify-tests.sh

# Build application
go build -o htCore main.go

# Run with default config
./htCore

# Or with custom config
./htCore --config=../Configurations/dev.json

API Testing

# Test version endpoint
curl http://localhost:8080/do \
  -H "Content-Type: application/json" \
  -d '{"action":"version"}'

# Expected response:
{
  "errorCode": -1,
  "errorMessage": "",
  "data": {
    "version": "3.0.0"
  }
}

Running Tests

# Comprehensive test suite
cd Application
./scripts/verify-tests.sh

# Quick tests
go test ./...

# With coverage
go test -cover ./...

# With race detection
go test -race ./...
